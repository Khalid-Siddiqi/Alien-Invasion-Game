# Alien Invasion Game
A Python Based Alien Invasion Game

## Description
The player controls the rocket ship at bottom centre of screen. The player can move the ship right and left using arrow keys and can shoot using space bar, after start of game a alien fleet moves across the screen player must try to stop it by firing at them if alien ship touches player ship player loses a ship of his own and if alien ship reaches bottom of screen player losses one ship if 3 ships are lost game over.
